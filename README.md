Crop Recommendation System

This project predicts the best crop based on soil nutrients and weather conditions using machine learning.

Model:
Random Forest is used for prediction.

Input:
Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, pH, Rainfall

Output:
Recommended crop

Evaluation Metrics:
The model is evaluated using Accuracy, Precision, Recall, and F1-score.

Model Comparison:
Different models like Logistic Regression, Decision Tree, KNN, and Random Forest were compared.
Random Forest gave the best performance.

Tuning:
Model performance is improved using GridSearchCV.
